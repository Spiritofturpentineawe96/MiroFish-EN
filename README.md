## 🔧 How it works

MiroFish-EN uses a swarm-style setup. That means several AI agents can work on the same task from different angles.

Typical flow:
1. You enter a prompt or prediction task.
2. The app sends the task to one or more AI agents.
3. The agents review the problem and return results.
4. The app combines the output into one view.
5. You compare the results and use them in your workflow.

This can help with:
- Forecasting
- Scenario testing
- Research notes
- Multi-model comparison
- Decision support

## 🔐 API keys

MiroFish-EN needs an API key for the provider you choose.

Keep your key private:
- Do not share it in chats or screenshots
- Do not post it in public folders
- Store it in the app settings only

If a key does not work:
1. Check that it was copied in full.
2. Make sure billing or access is active on the provider account.
3. Try another supported provider.

## 🧰 Troubleshooting

### App does not open
- Right-click the file and choose Run as administrator
- Make sure Windows did not block the download
- Check that the file finished downloading

### The app closes right away
- Open it again after restarting Windows
- Check whether your antivirus moved a file
- Make sure the folder still contains all app files

### AI requests fail
- Check your internet connection
- Confirm your API key is valid
- Make sure you selected a supported provider
- Try a different model name

### No output appears
- Wait for the request to finish
- Try a shorter prompt
- Restart the app and test again
